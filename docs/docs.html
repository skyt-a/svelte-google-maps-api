Getting Started

Welcome to the documentation for `svelte-google-maps-api`!

Installation

Install the library using your preferred package manager:


  pnpm install svelte-google-maps-api
  # or
  npm install svelte-google-maps-api
  # or
  yarn add svelte-google-maps-api

API Key Setup

You need a Google Maps API Key. Follow these steps:

  1. Go to the Google Cloud Console.
  2. Create a new project or select an existing one.
  3. Enable the "Maps JavaScript API".
  4. Enable any additional libraries needed by the components you plan to use (e.g., `marker`, `places`, `visualization`, `routes`). You can find the required libraries in the component documentation.
  5. Create an API Key under "Credentials".
  6. **Important:** Secure your API Key! Restrict it to your specific domains (for production) or IP addresses (for development) to prevent unauthorized use.

Basic Usage

Wrap your map components with the `APIProvider` and provide your API key.


<script lang="ts">
  import {
    APIProvider,
    GoogleMap,
    Marker
  } from 'svelte-google-maps-api';

  const apiKey = 'YOUR_GOOGLE_MAPS_API_KEY'; // Replace!

  const mapOptions = {
    center: { lat: 35.681, lng: 139.767 },
    zoom: 14
  };
</script>

<div style="height: 400px; width: 100%;">
  <APIProvider {apiKey} libraries={['marker']}>
    <GoogleMap {...mapOptions} mapContainerStyle="width:100%; height:100%;">
      <Marker position={mapOptions.center} />
    </GoogleMap>
  </APIProvider>
</div>

Now you can explore the documentation for specific components using the sidebar navigation.
