Simple Layers

These components toggle specific map layers on or off when they are mounted or unmounted.

TrafficLayer

Displays real-time traffic conditions. Corresponds to google.maps.TrafficLayer.

{#if showTraffic}  {/if}

TransitLayer

Displays the public transit network. Corresponds to google.maps.TransitLayer.

{#if showTransit}  {/if}

BicyclingLayer

Displays bicycle paths, suggested routes, and other overlays specific to bicycling. Corresponds to google.maps.BicyclingLayer.

{#if showBicycling}  {/if}

Props & Events (for all three)

Prop/EventTypeDescription
onLoad(layer: LayerInstance) => void (optional)Callback fired when the layer instance is created.
onUnmount(layer: LayerInstance) => void (optional)Callback fired before the layer is removed.

Note: These layers currently do not accept additional options via props after initialization.
