Polygon

Draws a polygon (a filled area with a border) on the map, corresponding to google.maps.Polygon.

Props

PropTypeDescription
pathsMVCArray<PathType> | PathType[]
(where PathType is MVCArray<LatLng> | (LatLng | LatLngLiteral)[])		Required. The vertices of the polygon, potentially including multiple paths for holes.
optionsgoogle.maps.PolygonOptions (optional)Additional polygon options (e.g., fillColor, strokeWeight, geodesic). See PolygonOptions docs.
draggableboolean (optional)If set to true, the user can drag this shape over the map.
editableboolean (optional)If set to true, the user can edit this shape by dragging the control points.
visibleboolean (optional)Whether the polygon is visible on the map.
Style Propsstring | number (optional)Direct props like fillColor, fillOpacity, strokeColor, strokeOpacity, strokeWeight, zIndex.
Polygon Events(...) => void (optional)Standard google.maps.Polygon events like onClick, onDblClick, onDragEnd, etc.
onPathsChanged(newPaths: MVCArray<MVCArray<LatLng>>) => void (optional)Custom event fired when the polygon's paths are changed.
onLoad(polygon: Polygon) => void (optional)Callback fired when the polygon instance is created.
onUnmount(polygon: Polygon) => void (optional)Callback fired before the polygon is removed.

Example


<GoogleMap {...mapOptions}>
  <Polygon
    paths={[
      [
        { lat: 35.690, lng: 139.765 },
        { lat: 35.688, lng: 139.770 },
        { lat: 35.690, lng: 139.775 },
        { lat: 35.692, lng: 139.770 }
      ] // Outer path
      // You can add inner paths (arrays of LatLngLiterals) here for holes
    ]}
    strokeColor="#FF0000"
    strokeOpacity={0.8}
    strokeWeight={2}
    fillColor="#FF0000"
    fillOpacity={0.35}
    editable={true}
  />
</GoogleMap>
