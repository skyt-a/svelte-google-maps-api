Marker

Renders a standard google.maps.Marker.

Props

PropTypeDescription
positionLatLng | LatLngLiteralRequired. The marker's geographical location.
optionsgoogle.maps.MarkerOptions (optional)Additional marker options (e.g., title, icon, draggable, label). See MarkerOptions docs.
Marker Events(...) => void (optional)All standard google.maps.Marker events are supported as props (e.g., onClick, onDragEnd, onPositionChanged).
onLoad(marker: Marker) => void (optional)Callback fired when the marker instance is created.
onUnmount(marker: Marker) => void (optional)Callback fired before the marker is removed from the map.

Example


<GoogleMap {...mapOptions}>
  <Marker
    position={ { lat: 35.681, lng: 139.767 } }
    title="Standard Marker"
    options={ { icon: '...', draggable: true } }
    onClick={ (e) => console.log('Marker clicked:', e) }
    onDragEnd={ (e) => console.log('Marker drag ended:', e.latLng?.toJSON()) }
  />
</GoogleMap>
