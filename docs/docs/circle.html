Circle

Draws a circle on the map, corresponding to google.maps.Circle.

Props

PropTypeDescription
centerLatLng | LatLngLiteralRequired. The center of the circle.
radiusnumberRequired. The radius in meters on the Earth's surface.
optionsgoogle.maps.CircleOptions (optional)Additional circle options. See CircleOptions docs.
draggableboolean (optional)If set to true, the user can drag this circle over the map.
editableboolean (optional)If set to true, the user can edit this circle by dragging the control points.
visibleboolean (optional)Whether the circle is visible.
Style Propsstring | number (optional)Direct props like fillColor, fillOpacity, strokeColor, strokeOpacity, strokeWeight, zIndex.
Circle Events(...) => void (optional)Standard google.maps.Circle events like onClick, onCenterChanged, onRadiusChanged, etc.
onLoad(circle: Circle) => void (optional)Callback fired when the circle instance is created.
onUnmount(circle: Circle) => void (optional)Callback fired before the circle is removed.

Example


<GoogleMap {...mapOptions}>
  <Circle
    center={ { lat: 35.678, lng: 139.775 } }
    radius={500} // meters
    strokeColor="#008000"
    strokeOpacity={0.8}
    strokeWeight={2}
    fillColor="#008000"
    fillOpacity={0.3}
    editable={true}
    draggable={true}
    onCenterChanged={ () => console.log('Circle center changed') }
    onRadiusChanged={ () => console.log('Circle radius changed') }
  />
</GoogleMap>
